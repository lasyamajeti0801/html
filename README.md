#  Static Website (HTML)

##  Description

This project is a static website developed using pure HTML. It focuses on structuring web content effectively and creating a clean, simple user interface without the use of advanced frameworks or backend technologies.

The project demonstrates the fundamentals of web development, including page layout, content organization, and basic navigation.

---

##  Features

* Structured web pages using HTML
* Simple and clean layout
* Easy navigation between pages
* Lightweight and fast loading
* Beginner-friendly design

---

##  Tech Stack

* HTML

---

## Purpose

This project was created to understand and demonstrate the basics of web development using HTML, including how web pages are structured and displayed in a browser.
